# Frontend para o teste prático VR Software

## Como rodar a aplicação

* ```yarn install && yarn dev```

## Como funciona

* O front é formado por duas tabelas, uma de alunos e outra de cursos. Para cada tabela, é possível adicionar um elemento novo, editar ou excluir um elemento. Além disso, é possivel vizualizar o elemento, e dentro de sua vizualização, vincular cursos ou alunos. É possível alternar entre darkmode e lightmode na navbar.
